What is Surveying Hub?

Surveying Hub is made by Joshua Dytianquin 👨‍💻, a 2nd-year BSCpE student at NU Fairview. Surveying Hub bridges the gap between classroom theory and real-world application for engineering students and trainees. Built as a multi-functional toolkit, the platform eliminates tedious manual formulas by offering instantaneous coordinate plotting, geometric calculations, and universal unit adjustments on a single streamlined interface.

Target Users

Surveying Hub is an all-in-one platform tailored for early-year Civil Engineering students and industry trainees. By integrating instant coordinate-based plotting with automatic area, volume, and unit conversion calculators, the platform eliminates tedious manual computations and the need for formula memorization. Featuring a streamlined, intuitive interface, it serves as both an educational resource for rapid estimation and a practical tool for real-world measurements.

FEATURES

Features of Surveying Hub

1 Smart Lot Analyzer

Allows users to input 3 or more dynamic (x, y) coordinate pairs to instantly generate a visual, plotted sketch of the polygon or lot shape. The built-in results panel systematically outputs the computed total area, total perimeter, and a comprehensive coordinates summary list for quick cross-checking and validation.

2 Basic Area Calculator

Provides rapid area assessments for standard geometric boundaries. Users can seamlessly calculate dimensions for shapes including Circles, Triangles, Rectangles, Squares, and Trapezoids by entering specific variables adapted directly to each configuration.

3 Volume Calculator

Computes three-dimensional measurements for essential geometric structures. Depending on the selected item—such as a Cube, Rectangular Prism, Cylinder, Sphere, or Cone—the program prompts users for layout parameters to calculate total volume capacities instantly.

4 Basic Unit Conversion

A specialized processing module designed for surveying-related units. Users select their required data matrix parameters to translate values accurately across fundamental engineering scales without executing manual calculations.
